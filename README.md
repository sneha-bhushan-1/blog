# ✍️ Sneha's Tech Blog

Welcome to the repository for my technical blog, deployed at [sneha-tech-blog.pages.dev](https://sneha-tech-blog.pages.dev/). 

This space is dedicated to exploring the intersection of enterprise software infrastructure, modern technical documentation methodologies, and user-centered design—specifically within the ITSM, ITAM, and Agentic AI landscapes.

---

## 👩‍💻 About Me

I am **Sneha Bhushan**, a full-time Technical Writer specializing in high-complexity enterprise SaaS products. 

My journey into technical communication is built on a deep, hands-on foundation in IT systems and operations. Having spent seven years working as an operating systems analyst and systems engineer specializing in Windows and VMware technologies, I have lived the lifecycle of IT service delivery. I spent years managing:
*   **Incidents & Problems** (Troubleshooting high-severity infrastructure issues)
*   **Change Management & Requests** (Deploying complex system upgrades and service fulfillment)
*   **Work Orders** (Translating operational tasks into execution)

This deep operational background allows me to write documentation not just from a product perspective, but from the perspective of the actual IT professionals, sysadmins, and support engineers who rely on it daily. Today, I combine this technical domain expertise with advanced UI/UX strategy to bridge the gap between complex backend systems and intuitive user documentation.

## 📚 Blog Content & Focus Areas

My writing focuses on the evolution of enterprise software, service automation, and documentation practices:

*   **ITSM, ITAM & Service Automation:** Insights into modern ticketing workflows, asset management, and scaling enterprise service desks.
*   **Agentic AI in Enterprise:** Exploring how AI-driven workflows and autonomous agents are transforming modern ITIL processes and user assistance.
*   **Docs-as-Code Methodology:** Practical guides on managing documentation using Git, GitHub, static site generators (Hugo/Jekyll), and continuous deployment pipelines (Cloudflare Pages).
*   **UI/UX & Documentation:** The silent bridge between microcopy, interface design, and technical accuracy to create friction-free user experiences.

## 🛠️ Built With

This blog is built using a modern **Docs-as-Code** toolchain for rapid, version-controlled publishing:

*   **Framework:** [Hugo](https://gohugo.io/) (Static Site Generator)
*   **Source Control:** Git & GitHub for content versioning and review
